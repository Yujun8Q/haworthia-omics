# Haworthia OMICS Demo

This read-only demo provides open-set phenotype matching, four-head attention
visualization, a numerical phenotype-neighbor network, and hierarchical clustering.

The model weights, training images, gallery images, checkpoints, and prototype catalog
are not included in this public Space repository and are not offered for download. The
runtime reads weights and numerical prototypes from a separate private model repository.

The demo is intended for academic research, education, and personal study. Its outputs
describe similarities in the current model and are not authoritative botanical
identifications or evidence of genetic relationships, hybridization, ancestry, or
evolutionary direction. The model can be wrong, and no identification, attention map,
phenotype network, neighbor relation, or clustering result should be used as decisive
evidence for research conclusions or practical decisions.

Uploaded images are not used for training or written to the project database. Platform
components may create short-lived temporary cache files while processing a request.
